# Works Tracker

This is a Next.js 16 application for managing construction/service works with:
- CRUD operations for works (projects) with dates
- Photo uploads for progress reporting
- Dashboard with statistics and charts
- Map integration with location markers

## Tech Stack

- Next.js 16 with App Router + Turbopack
- TypeScript
- Tailwind CSS + Shadcn/ui v4 (@base-ui/react)
- Supabase (PostgreSQL + Storage)
- Leaflet + OpenStreetMap for maps
- Recharts for charts
